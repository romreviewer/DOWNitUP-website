Windows Installation Guide

1

Download and locate the file

Locate the downloaded DOWNitUP-1.0.0.msi file in your browser's download bar or Downloads folder.

2

Launch the installer

Double-click the installer program to start the installation wizard.

3

Click "More Info"

When the blue Windows Defender SmartScreen window pops up warning that Microsoft Defender prevented an unrecognized app, click the small More info link directly under the warning text.

4

Select "Run anyway"

Once you click "More info", a new button labeled Run anyway will appear at the bottom right. Click it to proceed with setup.

5

Confirm system prompts

If Windows prompts you with User Account Control (UAC) to allow the installer to make changes, click Yes.

6

Complete installation

Follow the onscreen prompts in the setup wizard. The installer will place a shortcut on your desktop, and you can launch DOWNitUP instantly without any further warnings.
